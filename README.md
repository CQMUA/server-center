## 项目设置

```bash
npm install
```

### 编译和加载项目

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://github.com/element-plus/element-plus-vite-starter
cd element-plus-vite-starter
npm i
npm run dev
```

See `src/styles/element/index.scss`.
